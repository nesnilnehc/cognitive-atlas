Cognitive Atlas

帮助你更快找到、理解并连接有用认知对象的知识地图。

这是什么？

这里将模型、框架、原则、方法、工具等认知对象组织成一张可检索、可学习、可引用的知识地图。你可以先找问题，再看相关对象，并沿关联继续学习。

它如何组织内容：

  • 用于哪个阶段？感知 → 诊断 → 设计 → 执行 → 适应
  • 它更像什么？元层 → 概念层 → 结构层 → 程序层 → 工具层
  • 主要作用在哪？个体 → 团队 → 组织 → 生态 → 社会

适合快速查找认知对象、规划自我提升路径，也适合作为 AI 的结构化参考。
