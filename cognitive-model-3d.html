认知模型三维坐标系

用三维空间探索思维与决策模型：定位、比较、深入学习。

这是什么？

本站在三维坐标系中展示一批认知与决策模型（如 MECE、OKR、黄金圈等），按时间导向、控制深度、抽象程度定位，便于比较与延伸阅读。

三轴含义：

  • X 轴：过去 / 现在 / 未来（时间导向）
  • Y 轴：表达层 → 结构层 → 战略层 → 元认知层（控制深度）
  • Z 轴：工具 → 方法 → 原则 → 元模型（抽象程度）

适合做知识整理、战略思考或学习方法梳理时参考。
