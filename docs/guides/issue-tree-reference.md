# Issue Tree / Logic Tree 参考

**来源**：[Wikipedia - Issue tree](https://en.wikipedia.org/wiki/Issue_tree)（无中文维基，仅有英文）

---

## 别名

Issue tree 又称 **logic tree**（逻辑树）。

---

## 类型

| 类型 | 核心问题 | 用途 |
|------|----------|------|
| 诊断树 (Diagnostic tree) | why | 识别问题根因 |
| 方案树 (Solution tree) | how | 识别可行方案 |

---

## 四条规则（Chevallier / McKinsey）

1. 始终回答 why 或 how
2. 从左到右逐层深入
3. 分支需 MECE（相互独立、完全穷尽）
4. 洞察性拆解（insightful breakdown）

---

## 应用场景

- 管理咨询案例面试（如 market sizing）
- 市场估算
- 根因分析
- 方案设计
