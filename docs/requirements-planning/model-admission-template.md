## 3. 门槛校验 (G1-G5)
必须全部标注为“通过”才可继续流程：
- [ ] **G1 可复用性**: 至少能在 2 个以上独立场景复用。
- [ ] **G2 可解释性**: 能写出清晰定义、输入、输出和适用边界。
- [ ] **G3 可区分性**: 与库内已有模型非同义重复。
- [ ] **G4 可验证性**: 至少有 2 个可信来源或公认实践支撑。
- [ ] **G5 安全合规性**: 无明显误导或高风险。
