# Illustration 资产与多场景使用

## Problem / Need

每个认知对象配 illustration 便于用户理解；配图使用场景多（竖卡、落地页、学习手册、脚本模板等），需统一设计与来源追溯。用户希望在第一次接触（竖卡）时即有概念可视化，落地页和学习路径中可复用。

## Acceptance Criteria

- [ ] 竖卡支持可选 illustration，有则展示、无则跳过（MECE 为首例）
- [ ] 落地页/详情支持 illustration，与概念区块一起呈现
- [ ] 每个 illustration 附带 source 引用
- [ ] 设计规范：`docs/designs/illustration-design-spec.md`
- [ ] 使用场景设计：`docs/designs/2026-03-09-illustration-use-cases-design.md`

## Notes

- Phase 1 优先：竖卡 + 落地页
- Phase 2 可选：学习手册、脚本模板、OG 图
